API route

  • /api/v1/reviews?type=<type>&value=<value>

  • -value must be >= 0

  • -type must be = 'funny', 'cool', 'useful'

  • /api/v1/authors?author=<author>&stars_min=<value>&stars_max=<value>&date_min=<date>;&date_max=<date>

  • -author is required. Call with no author query for a list of authors

  • /api/v1/business?business=<business>&stars_min=<value>&stars_max=<value>&date_min=<date>;&date_max=<date>

  • -business is required. Call with no business query for a list of business
